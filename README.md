# Anypost
  A Social Media App 
</div>

## **Features -**

- Feed Posts
- Explore & Trending Posts
- Add & Delete Comment
- Add ,Update & Delete Post
- Follow / Unfollow
- User Profile Update
- Search User
- share post
- Like and Comment on Post
- Authentication:
  - User Signup
  - User Login
  - User Logout
 - Fully Responsive

## **Built with -**

- React JS
- useReducer  & useContext 
- React Router v6
